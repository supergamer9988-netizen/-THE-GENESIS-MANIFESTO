SECTION 4: PCB BOARD LAYOUT GUIDE ("River Flow" Architecture)
================================================================================

  +---------------------------------------------------------------------------+
  | [USB] [LDO]   [ESP32]   [Buffer1]    [CORE ARRAY]     [MUX] [Buffer2]   |
  |                                                                           |
  |               +---------+  +-------+  [Clip 00][Clip 08]  +------+      |
  |   [POWER]    |         |  |       |  [Clip 01][Clip 09]  |      |      |
  |               | ESP32   |  | Op-   |  [Clip 02][Clip 10]  | 4067 |      |
  |   + 5V IN     | WROOM   |  | Amp   |  [Clip 03][Clip 11]  |  IC  |      |
  |   |           |         |  | Drv.  |  [Clip 04][Clip 12]  |      |      |
  |  AMS1117      |  DAC25->|->|       |  [Clip 05][Clip 13]  |S0-S3 |      |
  |  (LDO)        |         |  +Bus--->|->ALL CLIPS            |->SIG |      |
  |   |           |  ADC36 <|<-|  Read |<-[Clip 14][Clip 06]  |      |      |
  |   +- 3.3V     +---------+  | OpAmp |  [Clip 15][Clip 07]  |      |      |
  |                             +-------+                       +------+      |
  |                                                                           |
  |         ===> AIRFLOW DIRECTION ===> (Cool side to Hot side)              |
  +---------------------------------------------------------------------------+

  Rules:
  - Keep Core Array (Clips) ISOLATED from ESP32 signals (no traces underneath)
  - Use COPPER POUR for full GND plane on both PCB layers
  - Place 0.1uF decoupling caps as close as possible to each IC VCC pin
  - Analog GND and Digital GND meet at ONE STAR POINT near LDO

SECTION 5: STEP-BY-STEP ASSEMBLY ORDER
================================================================================

  Step 1: Solder AMS1117-3.3 + Capacitors first (Power Stage)
  Step 2: Solder CD74HC4067 (16-Ch Mux IC)
  Step 3: Solder MCP6002 (Dual Op-Amp IC)
  Step 4: Solder ESP32 Module
  Step 5: Solder all 32 Fuse Clips (16 pairs) — this takes the most time
  Step 6: Wire Left Pins to Buffer Output Bus (Input Common Rail)
  Step 7: Wire Right Pins to Mux C0-C15 (Individual channels)
  Step 8: Wire Mux SIG -> Buffer 2 -> ESP32 VP (ADC)
  Step 9: Wire control lines S0-S3 -> ESP32 Pins 13,12,14,27
  Step 10: Flash genesis_self_test.ino -> Verify all 16 cores = [OK]
  Step 11: Flash genesis_production_blade.ino -> Ready for AI deployment

SECTION 6: QUICK REFERENCE (COMMAND SET)
================================================================================

  - V:128       Vectorize at 50% stimulus -> Returns DATA:x,x,x... (16 values)
  - V:0         Vectorize at 0% stimulus (baseline)
  - V:255       Vectorize at 100% stimulus (saturation)

  - SLEEP       Set DAC to 0, standby mode
